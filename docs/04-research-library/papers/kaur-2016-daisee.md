# P-006: DAiSEE — Engagement Recognition in the Wild

| Field | Value |
|-------|-------|
| **Authors** | Kaur et al. |
| **Year** | 2016 |
| **Tags** | AFF |
| **Label** | **[FACT]** |

## Dataset

9,068 video snippets, 112 users; labels: boredom, confusion, engagement, frustration (4 intensity levels each).

## Use

Benchmark for affective engagement classifiers in e-learning.

## Limitations for PedagogyX

- **Not physical classroom** layout
- Webcam close-ups ≠ distant classroom cameras
- **[HYPOTHESIS]** fine-tune transfers poorly without domain adaptation
