# P-003: Student Speaker Diarization in Group Discussion (EDM 2024)

| Field     | Value                                       |
| --------- | ------------------------------------------- |
| **Tags**  | CDA, EDM                                    |
| **Year**  | 2024                                        |
| **Label** | **[FACT]** (short paper in EDM proceedings) |

## Problem

Measuring **individual student talk time** in group work is labor-intensive.

## Method

Speaker diarization on noisy classroom audio; compare estimated vs human-annotated speaking time.

## Results **[FACT]**

- Diarization error rate ~34% on challenging classroom data
- Correlation ~0.62 between estimated and human student speaking time

## Limitations

- Group settings only; whole-class interactivity harder
- Privacy: storing voiceprints for students is sensitive

## PedagogyX Takeaway

Student talk metrics are **feasible but noisy**—must show confidence intervals and avoid high-stakes use.
