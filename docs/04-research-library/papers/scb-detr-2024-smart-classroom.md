# P-004: Multi-Scale Deformable Transformers for Student Behavior Detection

| Field     | Value               |
| --------- | ------------------- |
| **Year**  | 2024                |
| **arXiv** | 2410.07834          |
| **Tags**  | CV                  |
| **Label** | **[FACT]** preprint |

## Problem

Smart classroom CV suffers occlusion, blur, scale variation.

## Method

SCB-DETR: multi-scale deformable transformer detector for student behaviors.

## Results **[FACT]**

- mAP 0.626 (+1.5% vs baseline cited in paper)

## Limitations

- Lab/smart-classroom conditions may not transfer
- Behavior labels may not map to **pedagogical quality**
- Compute cost for district-wide 24/7 processing

## PedagogyX Takeaway

Use CV for **low-stakes descriptors** (hand raise, grouping) before **engagement scoring**.
