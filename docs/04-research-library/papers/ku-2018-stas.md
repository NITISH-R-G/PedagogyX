| Field       | Value                                                         |
| ----------- | ------------------------------------------------------------- |
| **Authors** | Ku, Chen, Liu, Wu, Chan, HABOOK Group                         |
| **Year**    | 2018                                                          |
| **Venue**   | ICCE 2018                                                     |
| **Tags**    | TE, SYS                                                       |
| **Code**    | Proprietary (HiTeach integration)                             |
| **Label**   | **[FACT]** published; metrics operational at scale per vendor |

## Problem

Manual lesson observation is expensive; teachers lack granular data for reflection.

## Method

Automatic collection of teaching behaviors via **smart classroom software** (not passive video only); generates integrated analytics report immediately after class.

## Metrics / Outputs

Teaching behavior categories tied to professional development workflows; precursor to **AI Sokrates five indexes** (T, P, C, D, E).

## Limitations

- Dependent on **HiTeach ecosystem** signals
- Generalization to non-digital whiteboard classrooms unclear
- Independent peer replication limited outside Taiwan deployments

## PedagogyX Takeaway

**Framework-linked automated reports** post-class drive adoption better than raw CV scores.
